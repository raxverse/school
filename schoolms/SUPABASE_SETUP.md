# 🗄️ Supabase Database Setup

## Create the `test` Table

The TestManager component expects a table called `test` in your Supabase database.

### **Option 1: Using Supabase Dashboard (Easy)**

1. Go to: `https://supabase.com`
2. Sign in with your account
3. Open your project: `pynplqrkmmmuksxcywzg` (from your Supabase URL)
4. Click **SQL Editor** (left sidebar)
5. Click **New Query**
6. Paste this SQL:

```sql
CREATE TABLE test (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  name TEXT NOT NULL,
  mobile BIGINT NOT NULL
);
```

7. Click **Execute** (or press Ctrl+Enter)
8. ✅ Table created!

---

### **Option 2: Using Supabase Table Editor (Fastest)**

1. Go to Supabase Dashboard
2. Click **Tables** (left sidebar)
3. Click **Create a new table**
4. Fill in:
   - Table name: `test`
   - Enable RLS: OFF (for demo purposes)
5. Click **Save**
6. Click **Insert column** to add:
   - Column name: `name` | Type: `text` | Required: ✓
   - Column name: `mobile` | Type: `int8` | Required: ✓
   - Column name: `created_at` | Type: `timestamptz` | Default: `NOW()` | Required: ✓

---

After creating the table:

1. Go back to your app: `http://localhost:3000`

3. Click **Test Supabase Connection** button

3. Click the `test` table
4. In **Table Editor**, click the **3 dots** → **Enable Realtime**
5. Select all columns
6. Click **Save**

## Row Level Security (RLS)

For production, enable RLS:

1. Click the `test` table
2. Click **RLS** tab
3. Enable RLS
4. Add policies:

```sql
-- Allow users to select their own data
CREATE POLICY "Users can select test" ON test
FOR SELECT USING (auth.uid() = user_id);

-- Allow users to insert data
CREATE POLICY "Users can insert test" ON test
FOR INSERT WITH CHECK (auth.uid() = user_id);

-- Allow users to update their data
CREATE POLICY "Users can update test" ON test
FOR UPDATE USING (auth.uid() = user_id);

-- Allow users to delete their data
CREATE POLICY "Users can delete test" ON test
FOR DELETE USING (auth.uid() = user_id);
```
